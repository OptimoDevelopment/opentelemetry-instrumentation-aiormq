OpenTelemetry aiormq Instrumentation
======================================

|pypi|

.. |pypi| image:: https://badge.fury.io/py/opentelemetry-instrumentation-aiormq.svg
   :target: https://pypi.org/project/opentelemetry-instrumentation-aiormq/

This library allows tracing requests made by the aiormq library.

Installation
------------

::

    pip install opentelemetry-instrumentation-aiormq
